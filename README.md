# Cavea-project

Small functional Inventories website made using Angular and typescript as backend.

1. backend:

   - go to /backend folder using cd and run these commands
   - run the command in terminal: npm install
   - on backend/.env change DB_PASSWORD value to your database password
   - run the code using: npm run dev.
   - you can seed the database with tables using: npm run seed.

2. frontend:

   - go to cd frontend/inventory-frontend
   - run the same command in terminal: npm install
   - run the code using: ng serve.
   - Note: run backend before frontend for website to fetch data and run them at the same time using different terminals.
